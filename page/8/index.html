C Trick

BASIC

底层是指针实现
a[5] == 5[a];
“ABCD”[2] == 2[“ABCD”] == ‘C’;
相当于(a + 5) == (5 + a)

result：4 and 5， 因为 sizeof 是编译时行为，运行时不会执行

MACRO

#pragma once是编译器相关的，有的编译器支持，有的编译器不支持，具体情况请查看编译器API文档，不过现在大部分编译器都有这个杂注了。#ifndef，#define，#endif是C/C++语言中的宏定义，通过宏定义避免文件多次编译。所以在所有支持C++语言的编译器上都是有效的，如果写的程序要跨平台，最好使用这种方式。
